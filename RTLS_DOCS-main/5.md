**No, there is no Reinforcement Learning in this codebase.**

## What's Actually Used

| Component | Learning Type | Method |
|-----------|---------------|--------|
| VAE Training | **Unsupervised** | Reconstruction loss (MSE) + KL divergence |
| Collision Classifier | **Supervised** | Binary cross-entropy on labeled data |
| Path Planning | **Optimization** | Gradient descent on loss function |

---

## Why It's Not RL

RL requires:
- Agent takes actions → receives rewards → learns policy

This system does:
- **No rewards** — just loss functions
- **No policy** — direct optimization of latent variable z
- **No episodes** — single forward optimization
- **No exploration/exploitation** — deterministic gradient descent

---

## What the Planning Actually Is

```python
# This is NOT RL — it's just gradient descent optimization
for step in range(max_steps):
    L_total = L_goal + λ_prior * L_prior + λ_collision * L_collision
    L_total.backward()
    optimizer.step()  # Update z directly
```

This is **gradient-based trajectory optimization** — finding a path by minimizing a cost function, not learning a policy through trial and error.
