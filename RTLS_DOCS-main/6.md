## 1.4 GECO in VAE Training

### Where GECO is Used

GECO is used in the **loss function** to automatically balance MSE and KL divergence.

### GECO Algorithm

```python
class GECO:
    def __init__(self, goal, step_size, alpha, geco_lambda_init, ...):
        self.goal = goal           # τ = 0.0008 (target MSE)
        self.step_size = step_size # 0.005
        self.alpha = alpha         # 0.95
        self.geco_lambda = geco_lambda_init  # 1
        self.cma = None            # Constraint Moving Average
    
    def loss(self, err, kld):
        # err = MSE, kld = KL divergence
        
        # Step 1: Compute constraint (how far from target)
        constraint = err - self.goal   # C = MSE - τ
        
        # Step 2: Update moving average (smooth the constraint)
        if self.cma is None:
            self.cma = constraint
        else:
            self.cma = (1 - α) * constraint + α * self.cma
        
        # Step 3: Compute update factor
        factor = exp(step_size × cma)
        
        # Step 4: Update λ
        self.geco_lambda = clamp(factor × self.geco_lambda, λ_min, λ_max)
        
        # Step 5: Return loss
        return kld + self.geco_lambda × err
```

### GECO Logic Table

| Condition | constraint | cma | factor | λ change | Effect |
|-----------|------------|-----|--------|----------|--------|
| MSE > τ (bad reconstruction) | + | + | > 1 | ↑ increases | Focus more on MSE |
| MSE < τ (good reconstruction) | - | - | < 1 | ↓ decreases | Allow more KL |
| MSE = τ (at target) | 0 | 0 | 1 | → stable | Balanced |

## 1.5 Full VAE Training Algorithm

```
╔══════════════════════════════════════════════════════════════════════════════╗
║                         VAE TRAINING ALGORITHM                               ║
╠══════════════════════════════════════════════════════════════════════════════╣
║                                                                              ║
║  INPUT:                                                                      ║
║    - Dataset D = {x₁, x₂, ..., xₙ} where xᵢ = [q₁..q₇, ee_x, ee_y, ee_z]    ║
║    - Hyperparameters: lr_vae, τ (g_goal), g_lr, g_alpha, epochs              ║
║                                                                              ║
║  INITIALIZE:                                                                 ║
║    - VAE model (encoder E, decoder D) with random weights θ                 ║
║    - Adam optimizer with lr = lr_vae                                        ║
║    - GECO: λ = 1, cma = None                                                ║
║                                                                              ║
║  ┌────────────────────────────────────────────────────────────────────────┐ ║
║  │ FOR epoch = 1 to epochs:                                               │ ║
║  │   ┌──────────────────────────────────────────────────────────────────┐ │ ║
║  │   │ FOR each batch B = {x₁, ..., xₘ} in DataLoader:                  │ │ ║
║  │   │                                                                  │ │ ║
║  │   │   # ═══════════ FORWARD PASS ═══════════                         │ │ ║
║  │   │                                                                  │ │ ║
║  │   │   # 1. Encode: x → (μ, σ)                                        │ │ ║
║  │   │   h = ELU(fc1(x))                                                │ │ ║
║  │   │   for layer in fc_encoder:                                       │ │ ║
║  │   │       h = ELU(layer(h))                                          │ │ ║
║  │   │   μ = fc21(h)                                                    │ │ ║
║  │   │   σ² = softplus(fc22(h)) + ε                                     │ │ ║
║  │   │   σ = √σ²                                                        │ │ ║
║  │   │                                                                  │ │ ║
║  │   │   # 2. Reparameterization trick: sample z                        │ │ ║
║  │   │   ε ~ N(0, I)                                                    │ │ ║
║  │   │   z = μ + ε × σ                                                  │ │ ║
║  │   │                                                                  │ │ ║
║  │   │   # 3. Decode: z → x'                                            │ │ ║
║  │   │   h = ELU(fc31(z))                                               │ │ ║
║  │   │   for layer in fc_decoder:                                       │ │ ║
║  │   │       h = ELU(layer(h))                                          │ │ ║
║  │   │   x' = fc41(h)                                                   │ │ ║
║  │   │                                                                  │ │ ║
║  │   │   # ═══════════ COMPUTE LOSSES ═══════════                       │ │ ║
║  │   │                                                                  │ │ ║
║  │   │   # MSE (reconstruction loss)                                    │ │ ║
║  │   │   MSE = (1/m) × Σᵢ ||xᵢ - x'ᵢ||²                                 │ │ ║
║  │   │                                                                  │ │ ║
║  │   │   # KL divergence                                                │ │ ║
║  │   │   KL = -0.5 × mean(Σⱼ (1 + log(σⱼ²) - μⱼ² - σⱼ²))                │ │ ║
║  │   │                                                                  │ │ ║
║  │   │   # ═══════════ GECO UPDATE ═══════════                          │ │ ║
║  │   │                                                                  │ │ ║
║  │   │   # Compute constraint                                           │ │ ║
║  │   │   C = MSE - τ                                                    │ │ ║
║  │   │                                                                  │ │ ║
║  │   │   # Update moving average                                        │ │ ║
║  │   │   cma = α × cma + (1-α) × C                                      │ │ ║
║  │   │                                                                  │ │ ║
║  │   │   # Update λ                                                     │ │ ║
║  │   │   κ = exp(g_lr × cma)                                            │ │ ║
║  │   │   λ = clamp(κ × λ, λ_min, λ_max)                                 │ │ ║
║  │   │                                                                  │ │ ║
║  │   │   # Final loss                                                   │ │ ║
║  │   │   Loss = KL + λ × MSE                                            │ │ ║
║  │   │                                                                  │ │ ║
║  │   │   # ═══════════ BACKWARD PASS ═══════════                        │ │ ║
║  │   │                                                                  │ │ ║
║  │   │   optimizer.zero_grad()                                          │ │ ║
║  │   │   Loss.backward()                                                │ │ ║
║  │   │   optimizer.step()     # Updates θ (model weights)               │ │ ║
║  │   │                                                                  │ │ ║
║  │   └──────────────────────────────────────────────────────────────────┘ │ ║
║  │                                                                        │ ║
║  │   # ═══════════ PERIODIC EVALUATION ═══════════                        │ ║
║  │   IF epoch % save_every == 0:                                          │ ║
║  │       Compute sample_consistency_posterior()                           │ ║
║  │       Compute sample_consistency_prior()                               │ ║
║  │       Compute AM_distances()                                           │ ║
║  │       Compute AM_AUC()                                                 │ ║
║  │       Save checkpoint (keep best by AM_AUC)                            │ ║
║  │                                                                        │ ║
║  └────────────────────────────────────────────────────────────────────────┘ ║
║                                                                              ║
║  OUTPUT:                                                                     ║
║    - Trained VAE model (encoder + decoder weights)                          ║
║    - Final GECO state (λ, cma)                                              ║
║                                                                              ║
╚══════════════════════════════════════════════════════════════════════════════╝
```

### How the data was generated:

```python
for i in range(100000):
    # 1. Sample random joint angles
    q = uniform(q_min, q_max)
    ee_xyz = FK(q)
    
    # 2. Sample random obstacle
    obs_x = uniform(0.2, 0.8)
    obs_y = uniform(-0.5, 0.5)
```

## 2.4 GECO in Classifier Training

### **GECO is NOT used in classifier training.**

The classifier uses simple **Binary Cross-Entropy (BCE) Loss**:

```python
def loss_function_obs(obs_logit, obs_label):
    bce_output = F.binary_cross_entropy_with_logits(obs_logit, obs_label, reduction='mean')
    return bce_output
```

Why no GECO?
- VAE training has two competing objectives (MSE vs KL) that need balancing
- Classifier training has only **one objective**: minimize BCE
- No need for adaptive weighting

## 2.5 Full Classifier Training Algorithm

```
╔══════════════════════════════════════════════════════════════════════════════╗
║                    COLLISION CLASSIFIER TRAINING ALGORITHM                   ║
╠══════════════════════════════════════════════════════════════════════════════╣
║                                                                              ║
║  INPUT:                                                                      ║
║    - Collision dataset D = {(xᵢ, obsᵢ, labelᵢ)}                             ║
║    - Pretrained VAE checkpoint                                               ║
║                                                                              ║
║  INITIALIZE:                                                                 ║
║    - Create VAEObstacleBCE model                                             ║
║    - Load pretrained VAE weights into encoder/decoder                        ║
║    - Create Adam optimizer for CLASSIFIER weights ONLY:                      ║
║        obs_params = [fc32, fc_obs[0], fc_obs[1], fc_obs[2], fc42]           ║
║        optimizer = Adam(obs_params, lr=0.0001)                               ║
║                                                                              ║
║  ┌────────────────────────────────────────────────────────────────────────┐ ║
║  │ FOR epoch = 1 to epochs_obs:                                           │ ║
║  │   ┌──────────────────────────────────────────────────────────────────┐ │ ║
║  │   │ FOR each batch (x, obs, label) in DataLoader:                    │ │ ║
║  │   │                                                                  │ │ ║
║  │   │   # ═══════════ FORWARD PASS ═══════════                         │ │ ║
║  │   │                                                                  │ │ ║
║  │   │   # 1. Encode x → z (using FROZEN encoder)                       │ │ ║
║  │   │   z, μ, logVar = encoder(x)                                      │ │ ║
║  │   │   # Note: encoder weights NOT updated                            │ │ ║
║  │   │                                                                  │ │ ║
║  │   │   # 2. Classify (z, obs) → logit                                 │ │ ║
║  │   │   h = fc32(concat(z, obs))      # [batch, 11] → [batch, 2048]    │ │ ║
║  │   │   h = ELU(h)                                                     │ │ ║
║  │   │   for layer in fc_obs:                                           │ │ ║
║  │   │       h = layer(ELU(h))         # [batch, 2048]                  │ │ ║
║  │   │   logit = fc42(ELU(h))          # [batch, 2048] → [batch, 1]     │ │ ║
║  │   │                                                                  │ │ ║
║  │   │   # ═══════════ COMPUTE LOSS ═══════════                         │ │ ║
║  │   │                                                                  │ │ ║
║  │   │   # Binary Cross-Entropy with Logits                             │ │ ║
║  │   │   # (sigmoid is applied internally)                              │ │ ║
║  │   │   BCE = -mean(label × log(σ(logit)) + (1-label) × log(1-σ(logit)))│ │ ║
║  │   │                                                                  │ │ ║
║  │   │   # ═══════════ COMPUTE ACCURACY ═══════════                     │ │ ║
║  │   │                                                                  │ │ ║
║  │   │   prob = sigmoid(logit)                                          │ │ ║
║  │   │   predicted = (prob >= 0.5)                                      │ │ ║
║  │   │   accuracy = (predicted == label).sum() / batch_size             │ │ ║
║  │   │                                                                  │ │ ║
║  │   │   # ═══════════ BACKWARD PASS ═══════════                        │ │ ║
║  │   │                                                                  │ │ ║
║  │   │   optimizer.zero_grad()                                          │ │ ║
║  │   │   BCE.backward()                                                 │ │ ║
║  │   │   # Gradients flow through encoder but DON'T update it           │ │ ║
║  │   │   # (encoder weights not in optimizer)                           │ │ ║
║  │   │   optimizer.step()  # Updates ONLY classifier weights            │ │ ║
║  │   │                                                                  │ │ ║
║  │   └──────────────────────────────────────────────────────────────────┘ │ ║
║  │                                                                        │ ║
║  │   # ═══════════ VALIDATION ═══════════                                 │ ║
║  │   Compute val_loss and val_accuracy on validation set                  │ ║
║  │                                                                        │ ║
║  │   # ═══════════ CHECKPOINTING ═══════════                              │ ║
║  │   IF epoch % save_every == 0:                                          │ ║
║  │       Save checkpoint                                                  │ ║
║  │       Keep best by val_accuracy                                        │ ║
║  │       Remove worst if exceeds num_best_ckpt                            │ ║
║  │                                                                        │ ║
║  └────────────────────────────────────────────────────────────────────────┘ ║
║                                                                              ║
║  OUTPUT:                                                                     ║
║    - Trained classifier (fc32 + fc_obs + fc42 weights)                      ║
║    - Encoder/decoder unchanged from pretrained VAE                          ║
║                                                                              ║
╚══════════════════════════════════════════════════════════════════════════════╝
```

## 3.1 Detailed Architecture

Planning uses the **trained VAE + Classifier** but doesn't train them. Instead, it **optimizes the latent variable z** directly.

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                          PLANNING SYSTEM ARCHITECTURE                           │
├─────────────────────────────────────────────────────────────────────────────────┤
│                                                                                 │
│  INPUTS:                                                                        │
│    q_start [1, 7]  - Start joint configuration                                 │
│    e_start [1, 3]  - Start EE position (FK of q_start)                         │
│    e_target [1, 3] - Target EE position                                        │
│    obstacles [{x,y,h,r}, ...] - List of obstacles                              │
│                                                                                 │
│  PRETRAINED MODELS (FROZEN):                                                    │
│    - VAE (encoder + decoder)                                                    │
│    - Collision Classifier                                                       │
│                                                                                 │
│  OPTIMIZED VARIABLE:                                                            │
│    z [1, 7] - Latent code (updated via gradient descent)                       │
│                                                                                 │
│  ┌─────────────────────────────────────────────────────────────────────────┐   │
│  │                       INITIALIZATION                                    │   │
│  │                                                                         │   │
│  │   1. Normalize start configuration                                      │   │
│  │      x_start = [q_start, e_start]                                       │   │
│  │      x_start_norm = (x_start - μ) / σ                                   │   │
│  │                                                                         │   │
│  │   2. Encode to get initial z                                            │   │
│  │      z_init = encoder(x_start_norm)[0]  # μ only, deterministic         │   │
│  │                                                                         │   │
│  │   3. Make z trainable                                                   │   │
│  │      z = z_init.clone().requires_grad_(True)                            │   │
│  │                                                                         │   │
│  │   4. Create optimizer                                                   │   │
│  │      optimizer = Adam([z], lr=planning_lr)                              │   │
│  └─────────────────────────────────────────────────────────────────────────┘   │
│                                      │                                          │
│                                      ▼                                          │
│  ┌─────────────────────────────────────────────────────────────────────────┐   │
│  │                     PLANNING LOOP (for max_steps)                       │   │
│  │                                                                         │   │
│  │   z [1, 7]                                                              │   │
│  │       │                                                                 │   │
│  │       ▼                                                                 │   │
│  │   ┌─────────────────────────────────────────────────────────────────┐   │   │
│  │   │ DECODER (FROZEN)                                                │   │   │
│  │   │                                                                 │   │   │
│  │   │   x_decoded_norm = decoder(z)                                   │   │   │
│  │   │   x_decoded = x_decoded_norm × σ + μ  (denormalize)             │   │   │
│  │   │   q_decoded = x_decoded[:, :7]                                  │   │   │
│  │   │   e_decoded = x_decoded[:, 7:10]                                │   │   │
│  │   └─────────────────────────────────────────────────────────────────┘   │   │
│  │       │                                                                 │   │
│  │       ├─────────────────┬─────────────────┬─────────────────┐           │   │
│  │       ▼                 ▼                 ▼                 │           │   │
│  │   ┌─────────┐     ┌─────────────┐   ┌─────────────────┐     │           │   │
│  │   │ L_goal  │     │ L_prior     │   │ L_collision     │     │           │   │
│  │   │         │     │             │   │                 │     │           │   │
│  │   │ ||e -   │     │ 0.5×||z||²  │   │ For each obs:   │     │           │   │
│  │   │  e_tgt||│     │             │   │   logit = cls(z)│     │           │   │
│  │   │         │     │ Keeps z     │   │   p = σ(logit/T)│     │           │   │
│  │   │ Distance│     │ near origin │   │   -log(1-p)     │     │           │   │
│  │   │ to goal │     │ (prior)     │   │                 │     │           │   │
│  │   └────┬────┘     └──────┬──────┘   └────────┬────────┘     │           │   │
│  │        │                 │                   │              │           │   │
```

## 3.4 GECO in Planning

### How GECO is Applied Differently

In **VAE training**, GECO balanced:
- MSE (reconstruction) vs KL (regularization)
- Single λ value

In **Planning**, GECO balances:
- L_prior vs L_goal → λ_prior
- L_collision vs L_goal → λ_collision
- **Two separate λ values**

### GECO Planning Algorithm

```python
# Initialize
lambda_prior = args.lambda_prior       # e.g., 0.002
lambda_collision = args.lambda_collision  # e.g., 0.15
C_prior_ma = None
C_collision_ma = None

for step in range(max_steps):
    # Compute losses
    L_goal = ||e_decoded - e_target||
    L_prior = 0.5 × ||z||²
    L_collision = Σ -log(1 - P(collision))
    
    if use_geco:
        # Compute constraint violations
        C_prior = L_prior - tau_prior_goal       # e.g., L_prior - 5.0
        C_collision = L_collision - tau_obs_goal  # e.g., L_collision - 0.5
        
        # Update moving averages (DIFFERENT α values!)
        if step == 0:
            C_prior_ma = C_prior
            C_collision_ma = C_collision
        else:
            C_prior_ma = 0.95 × C_prior_ma + 0.05 × C_prior      # Slow
            C_collision_ma = 0.4 × C_collision_ma + 0.6 × C_collision  # Fast
        
        # Update lambdas
        kappa_prior = exp(0.005 × C_prior_ma)
        kappa_collision = exp(0.005 × C_collision_ma)
        
        lambda_prior = clamp(kappa_prior × lambda_prior, 1e-6, 1000)
        lambda_collision = clamp(kappa_collision × lambda_collision, 1e-6, 1000)
    
    # Combined loss
    L_total = L_goal + lambda_prior × L_prior + lambda_collision × L_collision
    
    # Update z
    L_total.backward()
    optimizer.step()
```

### Why Different α Values?

| Constraint | α (smoothing) | Why |
|------------|---------------|-----|
| Prior (C_prior_ma) | 0.95 (slow) | Prior is stable, don't need fast adaptation |
| Collision (C_collision_ma) | 0.4 (fast) | Need to react quickly to collision events |

## 3.5 Full Planning Algorithm

```
╔══════════════════════════════════════════════════════════════════════════════╗
║                         PLANNING ALGORITHM                                   ║
║                    (Latent Space Optimization)                               ║
╠══════════════════════════════════════════════════════════════════════════════╣
║                                                                              ║
║  INPUT:                                                                      ║
║    - q_start [7]: Initial joint configuration                               ║
║    - e_start [3]: Initial EE position                                       ║
║    - e_target [3]: Target EE position                                       ║
║    - obstacles: List of [x, y, h, r]                                        ║
║    - Pretrained VAE and Classifier (frozen)                                 ║
║                                                                              ║
║  INITIALIZE:                                                                 ║
║    - λ_prior = lambda_prior (e.g., 0.002)                                   ║
║    - λ_collision = lambda_collision (e.g., 0.15)                            ║
║    - C_prior_ma = None                                                       ║
║    - C_collision_ma = None                                                   ║
║    - x_start = concat(q_start, e_start)                                     ║
║    - x_norm = (x_start - μ) / σ                                             ║
║    - z = encoder(x_norm)[μ].requires_grad_(True)                            ║
║    - optimizer = Adam([z], lr=planning_lr)                                  ║
║                                                                              ║
║  ┌────────────────────────────────────────────────────────────────────────┐ ║
║  │ FOR step = 0 to max_steps-1:                                           │ ║
║  │                                                                        │ ║
║  │   # ═══════════ FORWARD PASS (Through Frozen Models) ═══════════       │ ║
║  │                                                                        │ ║
║  │   # Decode current z                                                   │ ║
║  │   x_decoded_norm = decoder(z)                                          │ ║
║  │   x_decoded = x_decoded_norm × σ + μ                                   │ ║
║  │   q_decoded = x_decoded[:, :7]                                         │ ║
║  │   e_decoded = x_decoded[:, 7:10]                                       │ ║
║  │                                                                        │ ║
║  │   # ═══════════ COMPUTE LOSSES ═══════════                             │ ║
║  │                                                                        │ ║
║  │   # Loss 1: Goal reaching                                              │ ║
║  │   L_goal = ||e_decoded - e_target||                                    │ ║
║  │                                                                        │ ║
║  │   # Loss 2: Prior (stay near manifold)                                 │ ║
║  │   L_prior = 0.5 × ||z||²                                               │ ║
║  │                                                                        │ ║
║  │   # Loss 3: Collision avoidance                                        │ ║
║  │   L_collision = 0                                                      │ ║
║  │   FOR each obstacle obs:                                               │ ║
║  │       logit = classifier(z, obs)                                       │ ║
║  │       p_coll = sigmoid(logit / temperature)                            │ ║
║  │       L_collision += -log(1 - p_coll + ε)                              │ ║
║  │   L_collision /= num_obstacles                                         │ ║
║  │                                                                        │ ║
║  │   # ═══════════ GECO UPDATE (if enabled) ═══════════                   │ ║
║  │                                                                        │ ║
║  │   IF use_geco:                                                         │ ║
║  │       # Constraint violations                                          │ ║
║  │       C_prior = L_prior - τ_prior_goal                                 │ ║
║  │       C_collision = L_collision - τ_obs_goal                           │ ║
║  │                                                                        │ ║
║  │       # Update moving averages                                         │ ║
║  │       IF step == 0:                                                    │ ║
║  │           C_prior_ma = C_prior                                         │ ║
║  │           C_collision_ma = C_collision                                 │ ║
║  │       ELSE:                                                            │ ║
║  │           C_prior_ma = α_prior × C_prior_ma + (1-α_prior) × C_prior    │ ║
║  │           C_collision_ma = α_obs × C_collision_ma + (1-α_obs) × C_coll │ ║
║  │                                                                        │ ║
║  │       # Update lambdas                                                 │ ║
║  │       κ_prior = exp(α_geco × C_prior_ma)                               │ ║
║  │       κ_collision = exp(α_geco × C_collision_ma)                       │ ║
║  │       λ_prior = clamp(κ_prior × λ_prior, λ_min, λ_max)                 │ ║
║  │       λ_collision = clamp(κ_collision × λ_collision, λ_min, λ_max)     │ ║
║  │                                                                        │ ║
║  │   # ═══════════ TOTAL LOSS ═══════════                                 │ ║
║  │                                                                        │ ║
║  │   L_total = L_goal + λ_prior × L_prior + λ_collision × L_collision     │ ║
║  │                                                                        │ ║
║  │   # ═══════════ CHECK SUCCESS ═══════════                              │ ║
║  │                                                                        │ ║
║  │   IF L_goal < success_threshold:                                       │ ║
║  │       goal_reached = True                                              │ ║
║  │       BREAK                                                            │ ║
║  │                                                                        │ ║
║  │   # ═══════════ GRADIENT UPDATE ═══════════                            │ ║
║  │                                                                        │ ║
║  │   optimizer.zero_grad()                                                │ ║
║  │   L_total.backward()                                                   │ ║
║  │   optimizer.step()  # Updates z only                                   │ ║
║  │                                                                        │ ║
║  └────────────────────────────────────────────────────────────────────────┘ ║
║                                                                              ║
║  OUTPUT:                                                                     ║
║    - success: goal_reached AND collision_free                               ║
║    - q_final: decoder(z_final)[:, :7]                                       ║
║    - latent_path: [z₀, z₁, ..., z_final]                                    ║
║    - planning_time_ms: Time taken                                           ║
║                                                                              ║
╚══════════════════════════════════════════════════════════════════════════════╝
```
